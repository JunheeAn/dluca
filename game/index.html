Problem

Presence Gap / Uncanny Fatigue / Brand Incoherence

  • Presence Gap — 텍스트/음성만으로는 대면감과 신뢰 축적이 어렵다.
  • Uncanny Fatigue — 어긋난 표정·타이밍이 피로/거부감 유발.
  • Brand Incoherence — LLM 즉석 응답은 톤·금칙·지식 경계 일관성 붕괴.

→ 해결책: Visible Conversation — 보이는 대화와 대면 가능한 존재감.

Vision

대면 가능한 NPC — Core Experience

시선·표정·타이밍이 말과 동시에 움직인다. 플레이어는 “대답을 듣는다”를 넘어, 만난다를 경험한다.

Vision Video Key Visual
Core Loop

Listen → Think → Speak → React → Idle

침묵 구간에는 Back-channel Barks(끄덕/짧은 읊조림)로 불쾌부조화 최소화.

  • Listen(청취) — 사용자 입력(STT 포함) 수신
  • Think — LLM 스트림
  • Speak — TTS/오디오 합성
  • React — Lip-sync/Emotion/Gaze
  • Idle — Micro-Idle
Runtime Director

State Machine & Latency Budget

  • States: Idle ↔ Listening ↔ Thinking ↔ Speaking ↔ Reacting
  • Latency Budget: 자막/발화/표정 페이싱 동기
  • Back-channel Barks: 침묵 구간 긴장 완화
Orchestration

OnFinal / Coalesce Gate, Viseme & Emotion Smoothing

  • 합성 트리거: (a) OnFinal 수신, (b) 짧은 청크 Coalesce 임계 충족
  • Viseme attack/release/LPF + Emotion ease → Uncanny Fatigue 저감
  • UI Reveal / Back-channel로 침묵 구간 연결성 확보
