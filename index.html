Project LUA — LLM × MetaHuman

Local LLM × UE5 MetaHuman으로 Visible Conversation을 구현하는 실시간 디지털 휴먼 프로토타입.

Prototype Demo Vision Video EdTech Showcase 1-Page Executive Summary (PDF) About Me Portfolio Instagram

Problem

텍스트/음성 챗봇은 관계감·몰입감이 낮습니다. Visualization Gap, Uncanny Fatigue, 높은 운영 난이도가 한계입니다.

Solution

  • 브랜드 톤·표정·말투까지 반영하는 브랜드 맞춤형 리얼리즘 디지털 휴먼
  • LLM(Chat/STT) → TTS → LipSync(ARKit) → Emotion → (Next) Gaze Sync
  • 응답 속도·볼륨·채널·시선/립 동기화 프리셋

System (E2E)

Input → LLM(stream) → TTS → Viseme 계산 → LipSync + Emotion/Micro Idle → Subtitle/Audio Sync

Differentiators

  • Control & Orchestrability: 정책 기반 세밀 제어, Scenario Lock, Operator-in-the-loop
  • Persona & Scenario Templates: 말투·금칙·지식 경계를 프로필로 분리/교체
  • Visible Conversation UX: 립싱크·감정·미세동작으로 존재감/타이밍 강화

Roadmap (90 days)

  • M1: STT, Gaze Sync, 90–180s 루프
  • M2: PoC(1–2곳) 적용, 패키징 α, 로그/리플레이
  • M3: 패키징 β, 운영 미니패널, HCI Creative Award · Red Dot 제출
